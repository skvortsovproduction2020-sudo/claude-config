Александр предпочитает **old money** эстетику в дизайне. Его любимые палитры:

**01 — Ivory & Gold**
- Фон: #F4EEE4 (тёплый айвори)
- Акцент: #C9A84C (антикварное золото)
- Вторичный текст: #7A6E60 (тёплый тауп)
- Основной текст: #1A1410 (почти чёрный, тёплый)

**02 — Forest Green**
- Фон: #182218 (тёмный лесной зелёный)
- Акцент: #C9A84C (золото)
- Вторичный текст: #6A8A6A (приглушённый шалфей)
- Основной текст: #EDE4D4 (кремовый)

**03 — Mahogany Library**
- Фон: #1A120A (тёмный орех)
- Акцент: #C4A35A (тёплое золото)
- Вторичный текст: #9A8A70 (тёплый средний тон)
- Основной текст: #F0E4CC (пергамент)

**04 — Bordeaux Club**
- Фон: #180810 (тёмный бордо)
- Акцент: #C9A84C (золото)
- Вторичный текст: #8A6A72 (приглушённая роза)
- Основной текст: #F5EEE4 (слоновая кость)

**05 — Oxford Navy**
- Фон: #0C1628 (глубокий оксфордский синий)
- Акцент: #C9A84C (латунно-золотой)
- Вторичный текст: #5A7090 (сине-серый)
- Основной текст: #EEF2FF (холодный белый)

**Общие принципы:**
- Шрифт для заголовков: Cormorant Garamond (с засечками, элегантный)
- Шрифт для текста: Manrope
- Никакого корпоратива — личный бренд, heritage, аристократичность
- Золото вместо ярких акцентов
- Вторичный текст всегда достаточно светлый (#6A... и выше) для читаемости
