## Задача: Редизайн страницы /book

**Статус:** запланировано, ожидает ответов на вопросы

## Новая структура страницы /book

Три раздела:
1. 📅 **Расписание богослужений** — только чтение, просмотр календаря служб
2. ✦ **Записаться на таинство или требу** — Крещение, Венчание, Отпевание, Молебен, Освящение, Беседы
3. 🏠 **Пригласить священника на дом** — Причастие, Соборование, Выезд к больному

## Логика по типам

**Венчание** — слотовая запись:
- Показывать только даты, когда венчание разрешено по православному календарю
- На дате — доступные слоты времени, бронирование блокирует слот
- Статус: ожидает подтверждения → настоятель назначает священника

**Крещение** — гибкая запись:
- Календарь 30 дней вперёд, любой день
- Слоты: 12:00 (по договорённости), 13:00 (общее, несколько семей), 14:00–17:00 (индивидуальное)
- Выбор храма: большой / малый
- Также: беседа перед крещением

**Отпевание** — срочная заявка:
- Не слот, а форма → уведомление сразу ответственному священнику/администратору
- Свечницы в лавке могут вносить заявки (возможно новая роль)

**Беседы** — как текущая «личная встреча» у дежурного священника

## Ответы на вопросы (получены)

1. **Слоты — заданы системой, редактируются администратором**
   - Каждый обряд занимает ровно 1 час
   - Два места: Большой храм (БХ) и Малый храм (МХ)
   - Одновременно max 2 обряда (по одному в каждом храме)
   - **БХ слоты:** 12:00 (исключение, по договорённости), 13:00 (общее крещение по умолч.), 14–17:00 (индивидуальные)
   - **МХ слоты:** 10–17:00 (все индивидуальные, с 12:00 предпочтительно)
   - 13:00 МХ по умолчанию индивидуальное, но администратор может менять
   - Будни: 1 дежурный священник; сб/вс: все 4 священника

2. **Православный календарь для венчания** — автоматически по каноническим правилам (исключаем посты, Святки). Источник: patriarchia.ru или pravoslavie.ru

3. **Общее крещение (13:00 БХ)** — одновременно несколько семей, администратор может заменить на другой тип

4. **Свечницы / агенты** — могут получить доступ для бронирования отпевания (уточнить роль)

5. **Беседы** — у дежурных священников (как сейчас работает личная встреча)

## Горизонт записи
- Минимум 2 месяца, лучше 3 (венчание планируют сильно заранее)

## Дополнительные опции при записи
- **Крещение:** хор (+уведомление регенту), украшение купели цветами (+уведомление флористу)
- **Венчание:** хор, украшение храма цветами
- Богословское объяснение (что нужно знать перед таинством) — показывать перед формой

## Православный календарь (отдельный блок улучшений)
- Имена святых с правильными титулами (преподобный, священномученик, мученик...)
- Источник: patriarchia.ru или pravoslavie.ru / приложение «Православие»
- Клик по святому → разворачивается карточка с описанием
- Под святыми: чтения дня (Евангелие + Апостол)
- Возможно: краткие молитвы дня
