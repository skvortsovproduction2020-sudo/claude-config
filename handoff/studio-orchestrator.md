# Handoff — Studio Orchestrator (Главный штаб)

*Обновлено: 2026-06-22*

## 🎯 РОЛЬ

Главный штаб «Скворцов Продакшн». Отвечает за общую картину: какие зоны существуют, что в них висит, что делать прямо сейчас.

> **Как открыть:** написать «СЕО»

**Правило использования:**
1. Открыть эту сессию — посмотреть что висит и выбрать приоритет
2. Открыть отдельное окно Claude Code для нужной зоны
3. Работать параллельно в нескольких окнах одновременно
4. Вернуться сюда — зафиксировать что сделано, обновить очередь

## 🔄 КАК РАБОТАТЬ ПАРАЛЛЕЛЬНО

**Сейчас можно открыть несколько окон одновременно:**

```
Окно 1: Этот файл (штаб) — смотришь приоритеты
Окно 2: studio-cfo — финансовая модель
Окно 3: studio-cabinet — редизайн
Окно 4: любая другая зона
```

Каждое окно работает независимо. Задачи не мешают друг другу.

---

## ➡️ СЛЕДУЮЩИЙ ШАГ

1. Уточнить задачи из очереди (что конкретно нужно по Telegram, карусели, сбор данных)
2. По мере необходимости — создавать новые зоны (`studio-content`, `studio-crm`, и т.д.)
3. Возвращаться сюда чтобы не терять общую картину
