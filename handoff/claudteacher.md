### Параллельные сессии и совместная работа

**Вопрос:** можно ли открыть 2 сессии Claude Code одновременно для разных задач (требы + календарь)?
**Ответ:** да, технически работает. Риск — обе сессии могут одновременно перезаписать один handoff-файл.
**Решение для этого случая:** работать по очереди (договорились с Отцом Серафимом).

**Вопрос:** если Отец Серафим и Александр работают одновременно — будет ли конфликт?
**Ответ:** да, тот же баг только хуже — два разных компьютера. Правки могут перезаписать друг друга.
**Решение долгосрочное:** git-ветки (`feature/trebы`, `feature/calendar`). Пока не внедрили — работают по очереди.
